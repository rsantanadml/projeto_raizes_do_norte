# 🌵 Raízes do Nordeste

Sistema Front-end desenvolvido em **React + Vite** para gerenciamento de pedidos da rede **Raízes do Nordeste**.

O projeto foi desenvolvido como parte do Projeto Multidisciplinar do curso de Análise e Desenvolvimento de Sistemas.

## 🚀 Funcionalidades

* ✅ Login de usuários
* ✅ Cadastro de usuários
* ✅ Cardápio organizado por categorias
* ✅ Carrinho de compras
* ✅ Acompanhamento de pedidos
* ✅ Painel Administrativo
* ✅ Interface Responsiva

## 🛠 Tecnologias

* React
* Vite
* React Router
* Axios
* Tailwind CSS
* Context API

## 📦 Instalação

Clone o projeto:

```bash
git clone https://github.com/seuusuario/raizes-do-nordeste.git
```

Entre na pasta:

```bash
cd raizes-do-nordeste
```

Instale as dependências:

```bash
npm install
```

## ▶️ Executando

```bash
npm run dev
```

A aplicação estará disponível em:

```text
http://localhost:5173
```

## 🎯 Objetivo

O sistema foi desenvolvido para facilitar o processo de pedidos da rede Raízes do Nordeste, permitindo que clientes realizem pedidos de forma rápida enquanto a equipe administrativa acompanha as operações da empresa.
